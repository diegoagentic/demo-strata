# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-24

### Added
- Initial release of White Label Design System
- 30 production-ready components
- 201 design tokens with three-tier hierarchy
- Full TypeScript support
- Dark mode support
- Token-based theming system
- CVA (class-variance-authority) for variant management
- Comprehensive accessibility features
- Tree-shakeable exports
- Dual package exports (ESM + CJS)

### Components
- **Core UI**: Button, Input, Card, Badge, Modal
- **Forms**: Checkbox, Switch, Label, Textarea, Select, Radio, RadioGroup
- **Feedback**: Alert, Progress, Tooltip
- **Navigation**: Breadcrumb, Pagination
- **Data Display**: Table, Accordion, Avatar
- **Overlays**: Dropdown, Dialog, Popover, Toast
- **Layout & Utilities**: Separator, Tabs, Skeleton, Spinner, Slider, AspectRatio, ScrollArea

### Features
- ThemeProvider for easy customization
- useTheme hook for programmatic token access
- 50+ composable sub-components
- Full ARIA compliance
- Keyboard navigation support
- Focus management
- Responsive design
- Custom scrollbars
- Animated components
